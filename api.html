API DOCUMENTATION

We created an api for bridage . If you want to create your own site or mobile app you can use our api. This api return data in json format .

It have 3 endpoints

  1. for list of departments

    this endpoint return list of departments in json data

    > GET method https://kitsbrigade.herokuapp.com/api/departments/
