Architecture

Backend

we built backend using django ( open source python web framework ) . Django have ORM by using django orm we can create relation (tables) without writing raw SQL . By the way python comes with sqlite . So, we don't need install heavy database management system . SQLite is lightweight , We are planing to change to PostgreSQL as traffic gets high . we created three tables (departments,courses,units) using django orm (django orm write sql for us).

we built api using Django REST framework in around 25 lines of code . we used django admin panel for entering data in db . API have just three endpoints for departments, subjects , units . API returns data in json format .

Frontend

we firstly design site in Adobe xd . we just write html , css , javaScript . In javaScript we mainly created three functions . These functions call api (currently hosted in Heroku ) for json data and add these to DOM . while building these site we copy pasted almost from w3schools , MDN , django docs , djangorestframework docs ,of course from Stackoverflow . Only from Youtube we learnt everything for building these site .

Deployment

As we don't have budget we select only free options . For hosting Frontend we select Github Pages (because we already did once for csd note site ) here csd note site means a site we built previously for notes of data science branch . Our friends gave thier pdfs and book . we hosted them in github . Github pages is free . So we hosted fronted there
